Una sfera di centro C e raggio r è l'insieme di tutti i punti P che distano esattamente r dal centro C.

La distanza tra P e C è calcolata dal vettore (P - C), cioè |P - C|. Quindi:

```
|P - C| = r
```

Eleviamo al quadrato entrambi i lati (per evitare la radice quadrata).

```
|P - C|² = r²
```

Il quadrato della lunghezza è il dot product con se stesso:

```
(P - C) (P - C) = r²
```
