## Funzionamento

S = algoritmo di cifratura;
D = algoritmo di decifratura;
P = plain text;
K = key;
C = ciphered text;

```
	S = D
```

```
	C = S(P,k)
	P = D(C,k)
```

## Triade CIA

- C - Si
- I - No (server hash/HMAC a parte)
- A - No
